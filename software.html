Software

These tools emerged from the questions I was asking at each stage of my work.

From the Cell Biology Years

Built to analyze fluorescence anisotropy data from DNA damage experiments, and to wrangle proprietary microscope formats into something usable.

fai

Single-camera widefield fluorescence anisotropy imaging analysis. Processes parallel and perpendicular polarized emission via OptoSplit II.

Source Paper

microscoper

Data management and control library for Olympus IX83 microscope. Converts proprietary .vsi CellSense format to TIFF with metadata preservation.

Source

doodle

Interactive image manipulation library inspired by MATLAB's IMCROP. Core functionality contributed to scikit-image.

Source

From the Automation Turn

As experiments scaled from tens to thousands of cells, I needed systems that could orchestrate complex workflows across distributed instruments.

deepthought

Domain-driven microscopy control platform with integrated experiment planning. Layered architecture spanning Observation, Perception, and Biology domains.

Source bioRxiv arXiv

hard-link

Networking library for remotely connecting and controlling multiple microscopes, enabling distributed workflows and multi-instrument campaigns.

Source

deepthought-react

React-based web interface for microscope control with real-time visualization and experiment monitoring.

Source

From the Philosophy of Observation

What would software look like if centered around the sample's story rather than the instrument's capabilities?

anton

VLM-driven microscopy phenotype analysis using Cellular Microscopy Phenotype Ontology (CMPO). Combines traditional computer vision with vision language models.

Source arXiv

brushstrokes

Perception and observation architectures for intelligent image acquisition. Applies visual perception frameworks to automated microscopy.

Source

autotc

Digital twin modeling tissue culture facility workflows from vial preparation through cell monitoring.

Source

model

Data schema framework for active learning discovery systems integrating microscopes, sample preparation robotics, and hypothesis formulation engines.

Source

From Epistemic Microscopy

Remote collaboration with Janelia on next-generation microscopy systems for developmental biology.

gently

Control framework for DiSPIM (Dual-view Inverted Selective Plane Illumination Microscopy). Layered architecture with resumable acquisition plans and streamlined data management.

Source

subbu

Semantic microscope control framework enabling biologists to interact with microscopes using sample-centric concepts (embryos, not parameters).

Source

worm_VT

Vision Transformer for 4D lightsheet microscopy data of C. elegans embryo development. Factorized space-time attention with self-supervised pretraining.

Source

vlm_worm_test

Framework for evaluating Vision Language Model capabilities on lightsheet microscopy datasets. Automated embryo identification, lightsheet calibration, and focusing.

Source
